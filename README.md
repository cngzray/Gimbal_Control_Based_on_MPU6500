# Gimbal_Control_Based_on_MPU6500
基于 ESP32-C3 + MPU6500 实现双轴舵机云台稳定控制，采用卡尔曼滤波融合加速度计与陀螺仪姿态数据，实现云台跟随功能。

## 项目简介
本项目使用 ESP32-C3 主控，搭配 MPU6500 六轴姿态传感器，驱动两路模拟舵机分别控制 Pitch（俯仰）、Roll（横滚）轴，实现云台自稳。
核心特性：
1. I2C 驱动 MPU6500，内置设备地址扫描、传感器自检校准；
2. 加速度计静态姿态解算 + 陀螺仪积分，卡尔曼滤波姿态融合；
3. 开机自动静态校准，消除传感器零漂偏移；
4. 动态/静止状态自适应切换加速度计权重，高速运动时抑制加速度计干扰；
5. 角度区间线性映射至舵机 0~180°，带硬件限位保护；
6. 完整串口调试输出：原始加速度/陀螺仪、融合姿态、舵机映射角度、校准偏移；
7. 可直接适配 MPU6050（仅 WHO_AM_I 判断兼容）。

| 主控 | ESP32-C3 | Arduino 框架开发 |

| 舵机 ×2 | SG90/MG90S 模拟舵机 | 一路Pitch、一路Roll |

## 硬件接线定义
### I2C 传感器（MPU6500）
- SDA → GPIO4
- SCL → GPIO5
- VCC → 3.3V（**严禁接5V**）
- GND → GND

### 舵机信号线
- Pitch舵机（SERVO1）信号脚 → GPIO2
- Roll舵机（SERVO2）信号脚 → GPIO3
> 注意：舵机供电建议外接5V，避免ESP32板载LDO过载导致主控复位。

## 关键参数说明（可按需修改）
### 1. 引脚配置
```cpp
#define SERVO1_PIN 2    // Pitch舵机
#define SERVO2_PIN 3    // Roll舵机
#define MPU6500_ADDR 0x68
```
若传感器无响应，可切换地址 `0x69`。

### 2. 角度区间
```cpp
// 舵机物理输出区间
#define SERVO_MIN_ANGLE 0
#define SERVO_MAX_ANGLE 180
// 云台姿态物理限位
#define PITCH_MIN -90.0
#define PITCH_MAX 90.0
#define ROLL_MIN -90.0
#define ROLL_MAX 90.0
```

### 3. 卡尔曼滤波噪声参数
```cpp
float Q_angle = 0.001;    // 角度过程噪声
float Q_bias = 0.003;     // 陀螺仪零偏噪声
float R_measure = 0.03;   // 加速度计测量噪声
```
- 云台抖动大：适当增大 `R_measure`，降低加速度计权重；
- 长时间漂移严重：增大 `Q_bias`，加快陀螺仪零偏修正速度。

## 功能流程
1. **上电初始化**
   - 串口 115200 启动，等待USB CDC就绪；
   - I2C总线扫描，检测总线上所有I2C设备；
   - MPU6500 自检（WHO_AM_I寄存器校验），唤醒传感器退出睡眠；
2. **自动静态校准**
   - 采集100组静止加速度数据，计算Pitch/Roll零偏偏移；
   - 串口打印校准偏移量，标记校准完成；
3. **舵机初始化**
   - 两路舵机归中至90°中位；
4. **主循环控制（50ms周期）**
   1. 读取加速度、陀螺仪原始数据并转换物理单位；
   2. 静态姿态解算 + 卡尔曼滤波姿态融合；
   3. 将融合陀螺仪积分角度映射至舵机输出；
   4. 串口打印完整调试信息；

## 串口调试输出说明（波特率115200）
程序循环输出多组数据，方便调试与参数整定：
1. 加速度三轴原始值（单位g）
2. 陀螺仪三轴原始值（单位°/s）
3. 卡尔曼融合最优 Pitch/Roll 角度
4. 原始加速度解算角度、陀螺仪积分原始角度
5. 传感器校准状态与零偏移量
6. 陀螺仪角度映射后的舵机输出目标角度

## 代码核心模块介绍
1. **MPU6500 底层驱动**
   - I2C 单字节读写、16位寄存器读取；
   - WHO_AM_I 设备校验、总线扫描工具函数 `scanI2C()`；
2. **传感器自校准 calibrateSensor()**
   静止采集多组数据消除安装倾角偏移，无需手动调零；
3. **卡尔曼滤波 kalmanUpdate()**
   单轴通用卡尔曼实现，融合加速度静态角度与陀螺仪动态积分；
4. **姿态解算 calculateAngle()**
   - 通过重力向量解算静态Pitch/Roll；
   - 根据加速度模长判断运动/静止状态，动态切换加速度计可信度；
   - 输出滤波融合姿态与纯陀螺仪积分姿态；
5. **舵机映射与限幅**
   `mapPitchToServo` / `mapRollToServo` 将 ±90°姿态线性映射至0~180°舵机区间，并增加`constrain`硬件限位保护。
